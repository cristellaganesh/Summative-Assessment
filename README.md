# Summative-Assessment
📌 Project Overview

This project is a personal/creative portfolio website developed using HTML, CSS, and JavaScript. It showcases my skills in front-end development and design through multiple pages, including a homepage, about section, projects, and contact form! The site focuses on visual design, responsiveness, and interactivity all while integrating aesthetic as well.

🎯 Purpose

The purpose of this website is to:

- Present my personal brand and technical abilities
- Showcase selected design and development work
- Demonstrate responsive web development skills
- Provide a professional contact point

🧱 Site Structure

The website is organised into four main pages:

Home – Hero section with visual elements/stickers and an introduction
About – My personal background and skills overview
Projects – A display of recent selected work I have done
Contact – An interactive form with validation and confirmation modal

🛠️ Technologies Used
HTML – semantic structure
CSS – layout, styling, responsiveness, animations
JavaScript – interactivity (form validation, modal, theme toggle)
GitHub – version control and deployment

📱 Responsive Design Approach

Media queries were used to ensure compatibility across screen sizes. On mobile devices, layouts were simplified by stacking navigation elements, or simply removing them, resizing images, and removing or reducing decorative elements to improve usability and readability.

⚠️ Challenges Encountered

A key challenge was adapting complex desktop layouts (such as absolutely positioned decorative elements) for mobile screens. This required some restructuring of layouts in media queries to prevent overlapping while maintaining usability. Additional attention was required to ensure the consistent behaviour of interactive elements across devices.

💡 Best Practices Applied
- Separation of structure, style, and behaviour (HTML/CSS/JS)
- Responsive design using media queries
- User feedback through form validation and confirmation modal
- Optimised layout for multiple screen sizes
